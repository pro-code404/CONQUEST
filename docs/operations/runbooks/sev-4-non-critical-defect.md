# SEV-4 — Non-Critical Defect

**Severity:** SEV-4

## Process

1. Log in issue tracker with `correlation_id` if user-reported.
2. Triage within 2 business days.
3. Fix through normal PR workflow — no kill switches.
4. No customer notification unless SLA-impacting.

## Escalation

Escalate to SEV-3 if provider or auth impact emerges. Escalate to SEV-1 if any cross-tenant suspicion.
